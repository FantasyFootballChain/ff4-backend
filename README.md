# Backend for fantasy football chain

Server and oracle for fantasy football chain

## How to deploy
Setup DB credentials in `src/datasources/db.datasource.json`. Example:
```
{
  "name": "db",
  "connector": "mysql",
  "url": "",
  "host": "localhost",
  "port": 3306,
  "user": "username",
  "password": "password",
  "database": "db_name"
}
```
